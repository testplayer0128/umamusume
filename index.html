デビュー前

  1. 理事長がいるがまだ踏んでいない → 理事長と練習
  2. サポカが2人以上または友情トレがある → 料理を作る
  3. 絆上げができる練習がある → 練習へ
  4. 理事長とお出かけでき、3人以上の絆上げができない 
    → お出かけを検討

・友情トレができるサポカが1人以上or緑絆が3人以上 
 → 継続目安
・3人以下 → 諦めてもいいかも

 畑レベルを上げるときは野菜が3つ溜まっているとき

デビュー後(ジュニア期)

  1. 友情トレがある → 最優先で練習
  2. 絆上げができる練習がある → 優先的に練習
  3. お料理ポイントが2,000Pt or 2,500Ptたまった 
    → 以降は食材確保
  4. 体力90以下で練習も弱い 
    → 理事長とお出かけし野菜確保

・全員と友情トレができるor緑絆1人 ※理事長以外 
 → 継続目安
・青絆1人以上or緑絆2人以上 → 諦めたほうがいいかも

 畑レベルはにんにく以外Lv.2で終わる

クラシック期前半

  1. 友情トレがある → 最優先で練習
  2. 体力70以上でレースが緑野菜 → レースへ
  3. 体力70以下で理事長のお出かけが1~3回 → お出かけへ
  4. 体力70以下でお休みが緑野菜 → お休みへ
  5. 体力70以下で理事長のお出かけが4回目 → お出かけへ
  6. 賢さにサポカ2人以上で失敗率5%以下 → 賢さへ
  7. 体力70以上 → レースorサポカ2人以上の練習へ
  8. 上記以外 → お休みへ

 にんにくを5月前半にLv.2、以外をLv.3にする 
 畑レベルが足りない場合下振れ

 料理ポイントは4,000Pt以上を目標に

・料理を作るタイミング

 友情トレ2人以上or友情1人+サポカ1人以上
 体力80以下で失敗率10%以下なら友情1人でも賢さ料理を
 にんにく以外の在庫があふれる場合は友情1人でも

・料理を作らないタイミング

 友情トレなし
 にんじんが230個未満のスピード料理

クラシック期夏合宿

 賢さと他練習の強さ(メンバーの数)が同じ場合 → 賢さへ
※体力90以上or友情トレ発生中の野菜の在庫があふれる場合は賢さ以外でも可

・料理を作るタイミング

 友情トレ2人以上or友情1人+サポカ1人以上
 体力80以下で失敗率10%以下なら友情1人でも賢さ料理を
 にんにく以外の在庫があふれる場合は友情1人でも

・料理を作らないタイミング

 友情トレなし
 にんじんが180個未満のスピード料理

 料理ポイントは6,000Pt以上を目標に、最低でも5,500Pt

クラシック期後半

 夏合宿終了後すべての畑をLv.3にする

・料理を作るタイミング

 友情トレ2人以上or友情1人+サポカ1人以上
 体力80以下で失敗率10%以下なら友情1人でも賢さ料理を
 にんにく以外の在庫があふれる場合は友情1人でも

・料理を作らないタイミング

 友情トレなし
 にんじんが230個未満のスピード料理

 料理ポイントは8,000ptでシニアへ

シニア期前半

・料理を作るタイミング

 友情トレ2人以上or友情1人+サポカ1人以上
 体力80以下で失敗率10%以下なら友情1人でも賢さ料理を
 にんにく以外の在庫があふれる場合は友情1人でも

・料理を作らないタイミング

 友情トレなし
 にんじんが230個未満のスピード料理※800料理限定

 6月前半までにLv.4を3つ、試食会時にLv.4を4つにする
 料理ポイントは10,400Pt以上で夏合宿へ

シニア期夏合宿

 賢さと他練習の強さ(メンバーの数)が同じ場合 → 賢さへ
※体力90以上or友情トレ発生中の野菜の在庫があふれる場合は賢さ以外でも可

  1. 友情トレがある → 最優先で練習
  2. 体力70以上でレースが緑野菜 → レースへ
  3. 体力50以上で賢さにサポカが2人以上 → 賢さへ
  4. 体力80以上でサポカ2人以上 
    → 1番ステータスが伸びる練習へ
  5. 体力90以上 → レースへ
  6. 上記以外 → お休みへ

・800料理を作るタイミング

 友情トレ2人以上or友情1人+サポカ1人以上
 体力80以下で失敗率10%以下なら友情1人でも賢さ料理を
 にんにく以外の在庫があふれる場合は友情1人でも

・800料理を作らないタイミング

 友情トレなし
 にんじんが270個未満のスピード料理

 12,000Ptを超えると250料理や500料理も選択肢に入れる
 250料理と500料理は次に800料理が作れる野菜を残すこと 
 ※難しい場合は800料理のみでOK

シニア期後半

  1. 友情トレがある → 最優先で練習
  2. 体力80以上でレースが緑野菜 → レースへ
  3. 理事長のお出かけが5回未満 → お出かけへ
  4. 賢さにサポカ2人以上で失敗率5%以下 → 賢さへ
  5. 体力80以下でお休みが緑野菜 → お休みへ
  6. 体力70以上 → レースorサポカ2人以上の練習へ
  7. 上記以外 → お休みへ

 畑レベルはサポカに合わせて上げる

・G1プレート用野菜目安

 編成しているサポカの野菜は2種類までなら80個ほど少なくてもOK

・800料理を作るタイミング

 友情トレ2人以上or友情1人+サポカ1人以上
 体力80以下で失敗率10%以下なら友情1人でも賢さ料理を
 にんにく以外の在庫があふれる場合は友情1人でも

・800料理を作らないタイミング

 友情トレなし
 にんじんが270個未満のスピード料理
 体力90以上で友情1人+サポカ1人かつ練習の野菜が300個以下
 ※250料理や500料理の作成は可

URA(ファイナル)

・野菜目安

 上記の数がある場合シニア後半での料理不足となるため
 若干の下振れとなる

 また、6回目のG1プレート作成後、200個以上余る野菜が
 3種類以上ある場合もシニア後半での料理不足となる
