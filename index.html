デビュー前

  1. 理事長がいるがまだ踏んでいない → 理事長と練習
  2. サポカが2枚以上または友情トレがある → 料理を作る
  3. 絆上げができる練習がある → 練習へ
  4. 友情トレができるサポカが1枚以上or緑絆が3人以上 
    → 継続目安
    3人以下 → 諦めてもいいかも
  5. 理事長とお出かけでき、3人以上の絆上げができない 
    → お出かけを検討

 畑レベルを上げるときは野菜が3つ溜まっているとき

デビュー後(ジュニア期)

  1. 友情トレがある → 最優先で練習
  2. 絆上げができる練習がある → 優先的に練習
  3. お料理ポイントが2000Ptたまった  → 以降は食材確保
  4. 体力が90以下で練習も弱い 
    → 理事長とお出かけして野菜確保
  5. 全員と友情トレができるor緑絆1人 ※理事長以外 
    → 継続目安
    青絆1人以上or緑絆2人以上 → 諦めたほうがいいかも

 畑レベルはにんにく以外Lv.2で終わる

クラシック期前半

  1. 友情トレがある → 最優先で踏む
  2. 体力70以上かつレースが緑野菜 → レースへ
  3. 体力70以下かつ理事長のお出かけが1~3回 → お出かけへ
  4. 体力70以下かつお休みに緑野菜 → お休みへ
  5. 体力70以下で理事長お出かけが4回目 → お出かけへ
  6. 賢さにサポカ2枚以上かつ失敗率5%以下 → 賢さへ
  7. 体力70以上 → レースorサポカ2枚以上の練習
  8. 上記以外 → お休みへ

 にんにくを5月前半にLv.2、夏前ににんにく以外Lv.3にする 
 畑レベルが足りない場合下振れ 料理ポイントは4000Pt

・料理を作るタイミング

 友情トレ2人以上or友情1人+サポカ1人以上
 体力80以下で失敗率10%以下なら友情1人でも
 にんにく以外の在庫があふれる場合は友情1人でも

・料理を作らないタイミング

 友情トレなし
 にんじんが230個未満のスピード料理

クラシック期夏休み

 賢さと他練習の強さ(メンバーの数)が同じ場合 → 賢さへ
※体力90以上or友情トレ発生中の野菜の在庫があふれる場合は賢さ以外も可

・料理を作るタイミング

 友情トレ2人以上or友情1人+サポカ1人以上
 体力80以下で失敗率10%以下なら友情1人でも
 にんにく以外の在庫があふれる場合は友情1人でも

・料理を作らないタイミング

 友情トレなし
 にんじんが180個未満のスピード料理

 料理ポイントは6000Pt以上を目標に、最低でも5500Pt
